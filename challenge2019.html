Challenge for Objects365 is proposed to have two tracks: Full Track and Tiny Track. The goal of Full Track is to explore the upper-bound performance of object detection systems, given all the 365 classes and 600K training images. 68K images are used for validation and another 70K images are used for testing. To evaluate the performance of object detection, the evaluation criteria for COCO (IOU from 0.5 to 0.95) benchmark will be adopted. For the Tiny Track, 100K images from the complete set of database is available to train the algorithms for the task of object detection with COCO 80 categories. The track is designed to encourage the novel methods which are not data-hungry. Also, it is friendly to researchers in the academic without large computational infrastructure. The same 10K images are used for validation and another 50K images are used for testing but only on the COCO 80 categories.
